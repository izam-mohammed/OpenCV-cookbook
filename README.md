## 🚀 Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/OpenCV-cookbook.git
   ```

2. Navigate to the project directory:

   ```bash
   cd OpenCV-cookbook
   ```

3. Explore the files & folders and dive into each topic
